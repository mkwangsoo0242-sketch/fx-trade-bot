# AUD/USD 5개년 마스터 전략 [The Iron Shield]

본 폴더에는 AUD/USD 통화쌍의 5개년(2021-2025) 통합 분석을 통해 완성된 최적의 고수익 전략이 포함되어 있습니다.

## 1. 전략 개요
- **이름**: The Iron Shield (Mega Risk 5%)
- **핵심 로직**: 평균 회귀 (Mean Reversion)
- **주요 지표**: 200 EMA(장기 추세), RSI(과매수/과매도)
- **리스크 관리**: 거래당 5% 복리(Compounding) 적용

## 2. 권장 파라미터
- **시점**: EMA 200과 괴리가 크고 RSI가 30/70일 때 반전 매매
- **손익비**: 익절 30핍 / 손절 60핍 (승률 70% 이상 확보를 위한 최적의 비율)
- **레버리지**: 100배
- **초기 자본**: $1,000

## 3. 백테스트 결과 (2021-2025)
- **누적 수익률**: **+346.15%**
- **최대 낙폭(MDD)**: **33.80%**
- **최종 자산**: **$4,461.45** (복리 적용 시)

## 4. 실행 방법
```bash
python audusd_iron_shield_mega.py
```
